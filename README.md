RFID-Based Employee Attendance System (ARM LPC2129)

📌 Overview
This project implements a secure RFID-based employee attendance system using ARM7 LPC2129 microcontroller. It uses UART communication for RFID data, I2C for RTC (DS1307), and a Linux backend for logging attendance.

⚙️ Features
RFID-based identification (EM-18)
Real-time clock integration (DS1307)
LCD display for live status
UART interrupt-based data handling
Linux backend for attendance logging

🧰 Technologies Used
Embedded C
ARM7 LPC2129
UART, I2C Protocols
Linux Serial Communication

🔌 Hardware Components
LPC2129 Development Board
EM-18 RFID Reader
DS1307 RTC Module
16x2 LCD Display

🖥️ Software Components
Keil uVision (Embedded C)
Linux (Serial Communication Logging)

📊 Output
Displays time and date on LCD
Logs employee IN/OUT time
Sends data to Linux system
